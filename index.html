How To Install Phix

Via PEAR

The best way to install Phix is via the standard PHP pear command.

Installing On Debian Or Ubuntu

Install phix on either Debian or Ubuntu using the PEAR installer:

# binary modules that may be missing
sudo apt-get install php5-xdebug
sudo apt-get install php5-imagick
sudo apt-get install php5-xsl

# make sure we have the latest PEAR installer
sudo pear upgrade pear/pear

# install phix
sudo pear channel-discover pear.phix-project.org
sudo pear -D auto_discover=1 install -Ba phix/phix4componentdev

Installing On Fedora, CentOS, or RedHat Enterprise Linux

Install phix on Fedora, CentOS, RedHat Enterprise Linux or other RPM-based Linux distros using the PEAR installer:

# binary modules that might be missing
sudo yum install php-process
sudo yum install php-pecl-imagick
sudo yum install php-pecl-xdebug

# make sure we have the latest PEAR installer
sudo pear upgrade pear/pear

# install phix
sudo pear channel-discover pear.phix-project.org
sudo pear -D auto_discover=1 install -Ba phix/phix4componentdev

What Gets Installed

This installs the phix command itself, plus all the tools that phix automatically supports for generating code review information, running unit tests, and generating documentation.
